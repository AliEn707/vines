Install Ruby 1.9.3

Before we can install the Vines gem, we need to install Ruby 1.9.3. The easiest way to do this is with RVM: Ruby Version Manager.

The install is only two steps:

  1. Install the OS libraries upon which Ruby depends.
  2. Install Ruby using RVM.

Mac OS X

Install Xcode from the Mac App Store. Then make sure to install the Command Line Tools package from the Xcode > Preferences > Downloads screen. This installs all of the libraries and compiler tools needed to build Ruby.

Ubuntu

Install the following packages using apt-get. Ubuntu 12.04 provides a Ruby 1.9.3 package, so we can skip the RVM steps. 

$ sudo apt-get -y install build-essential bison openssl \
  libreadline6 libreadline6-dev curl git-core zlib1g zlib1g-dev \
  libssl-dev libyaml-dev libsqlite3-0 libsqlite3-dev sqlite3 \
  libxml2-dev libxslt-dev autoconf libc6-dev ncurses-dev ruby1.9.3
$ sudo gem install vines

Fedora

Install the following packages using yum. Fedora 17 provides a Ruby 1.9.3 package, so we can skip the RVM steps. 

$ sudo yum install -y bash curl git gcc-c++ patch readline \
  readline-devel zlib zlib-devel libyaml-devel libffi-devel \
  openssl-devel make bzip2 sqlite sqlite-devel libxslt \
  libxslt-devel libxml2 libxml2-devel ruby-devel
$ sudo gem install vines

SUSE

Install the following packages using zypper. 

$ sudo zypper install -y bash curl git gcc-c++ patch \
  libreadline6 readline-devel zlib zlib-devel libyaml-devel \
  libffi45-devel libopenssl-devel make bzip2 sqlite3 \
  sqlite3-devel libxslt libxslt-devel libxml2 libxml2-devel

FreeBSD

Install the following packages using pkg_add. After installing the packages, run bash to enter a bash shell before installing RVM. 

$ sudo pkg_add -r bash curl git openssl bzip2 readline \
  sqlite3 libxml2 libxslt libyaml
$ bash

Install RVM & Ruby 1.9.3

With the OS libraries installed, we can now use RVM to install Ruby. 

$ bash < <(curl -s https://rvm.beginrescueend.com/install/rvm)
$ echo '[[ -s "$HOME/.rvm/scripts/rvm" ]] && . "$HOME/.rvm/scripts/rvm" # Load RVM function' >> ~/.bash_profile
$ source .bash_profile
$ rvm install 1.9.3
$ rvm use 1.9.3 --default
$ gem update --system
$ gem install vines

That's all there is to it! RVM can manage not just Ruby versions, but also gemsets for each app. Read the RVM documentation for more advanced uses.
