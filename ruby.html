Install Ruby 1.9.2

Before we can install the Vines gem, we need to install Ruby 1.9.2. The easiest way to do this is with RVM: Ruby Version Manager.

The install is only two steps:

  1. Install the OS libraries upon which Ruby depends.
  2. Install Ruby using RVM.

Mac OS X

Install Xcode from the Mac App Store. This installs all of the libraries and compiler tools needed to build Ruby.

Ubuntu

Install the following packages using apt-get. 

$ sudo apt-get -y install build-essential bison \
  openssl libreadline6 libreadline6-dev curl \
  git-core zlib1g zlib1g-dev libssl-dev libyaml-dev \
  libsqlite3-0 libsqlite3-dev sqlite3 libxml2-dev \
  libxslt-dev autoconf libc6-dev ncurses-dev

Fedora

Install the following packages using yum. 

$ sudo yum install -y bash curl git gcc-c++ patch \
  readline readline-devel zlib zlib-devel \
  libyaml-devel libffi-devel openssl-devel make \
  bzip2 libxslt libxslt-devel libxml2 libxml2-devel

SUSE

Install the following packages using zypper. 

$ sudo zypper install -y bash curl git gcc-c++ \
  patch libreadline6 readline-devel zlib zlib-devel \
  libyaml-devel libffi45-devel libopenssl-devel \
  make bzip2 libxslt libxslt-devel libxml2 \
  libxml2-devel

Install RVM & Ruby 1.9.2

With the OS libraries installed, we can now use RVM to install Ruby. 

$ bash < <(curl -s https://rvm.beginrescueend.com/install/rvm)
$ echo '[[ -s "$HOME/.rvm/scripts/rvm" ]] && . "$HOME/.rvm/scripts/rvm" # Load RVM function' >> ~/.bash_profile
$ source .bash_profile
$ rvm install 1.9.2
$ rvm use 1.9.2
$ gem install vines

That's all there is to it! RVM can manage not just Ruby versions, but also gemsets for each app. Read the RVM documentation for more advanced uses.
